ODXVR x NTS: Collaborative exploration of rich corpus data using immersive virtual reality and non-immersive technologies

About

This web page presents collected data of a user interaction study the VRxAR Labs research group conducted during May 2019. Within the study, eight pairs of participants (n = 16) operated our developed collaborative Immersive Analytics system, consisting of an immersive Virtual Reality application and a non-immersive desktop application, in order to collaboratively (synchronous, co-located) investigate the usage of hashtags in regards to language variability on Twitter.

Logging: 3D visualizations

Based on the implemented logging system in both applications, we created 3D visualizations of the participants' travel movements using the R programming language. The X/Y/Z axes map to Latitude/Longitude/Time.
